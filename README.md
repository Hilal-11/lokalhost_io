## What is Lokallhost.io?

Lokallhost.io is not just another component library — it's a **full-stack development ecosystem** built for developers, designers, agencies, startups, and businesses who want to ship faster without compromising quality.

From AI application UI kits to cross-platform mobile assets, from animated web templates to end-to-end software development services — everything you need to build, launch, and scale modern digital products lives here.

Production-ready components and section blocks purpose-built for **AI application development** and modern web experiences.

## Getting Started

### Prerequisites

- Node.js `18.17+`
- npm / yarn / pnpm / bun

### Installation

```bash
# Clone the repo
git clone https://github.com/your-org/lokallhost.git
cd lokallhost

# Install dependencies
npm install
# or
pnpm install
```

### Development Server

```bash
npm run dev
# or
pnpm dev
# or
yarn dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm run start
```

## Project Structure

```
lokallhost/
├── app/                    # Next.js App Router
│   ├── (auth)/             # Auth routes
│   ├── (dashboard)/        # Dashboard routes
│   └── (marketing)/        # Public marketing pages
├── components/
│   ├── ui/                 # Core UI primitives
│   ├── blocks/             # Section-level blocks
│   ├── ai/                 # AI-specific components
│   └── shared/             # Shared layout components
├── templates/              # Full page templates
├── kits/                   # Auth & SaaS starter kits
├── mobile/
│   ├── react-native/       # RN components & screens
│   └── flutter/            # Flutter widgets & screens
├── design-kits/            # Figma export tokens & assets
├── lib/                    # Utilities, hooks, configs
├── styles/                 # Global styles & design tokens
└── public/                 # Static assets
```
